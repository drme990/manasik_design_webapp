# Manasik Design App - Next.js

A web-based design editor for the مناسك Foundation's marketing operators.

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

- `app/` - Next.js App Router pages
- `components/` - Reusable React components
- `lib/` - Utility functions and state management
- `types/` - TypeScript type definitions

## Features

- Design editor with canvas, layers, and transformations
- Text, image, and shape layers
- Multi-image collage support
- PDF generation and assembly
- Booking template system
- RTL support for Arabic
- Offline-first with IndexedDB

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- IndexedDB
